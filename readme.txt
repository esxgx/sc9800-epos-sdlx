关于开发，请参看test里的简单实例。

////////////////////////////////////////////////////////////////////
SDLX 是 SDL + X(目前指EPOS) 的合写

使用SDLX开发时选择部分RVDS的标准库, 部分则重新实现.
X_inc内包含RVDS标准库里没有的函数

调试时在源代码上加入#pragma import(__use_no_semihosting_swi)
关闭半系统调试，如果一切OK直接刷机即可。

如果不能编译通过(如缺少符号等)，请联系Sochip取得SDLX更新包.
////////////////////////////////////////////////////////////////////

另外, 如果要使程序运行起来，必须且仅在在某份源代码里包含：
#include "_axf.h"
最好像test一样单独建一个magic.c, axf.h内的入口点不得修改。


然后编写main()函数, 注意argv[0]目前无定义, argv[1]是参数输入。

任何问题，请联系esxgx@163.com或者Sochip公司获取最新资讯。
